#pinout

- **pins**:
 - GP0 = step0 (pin 1)
 - GP1 = dir0 (pin 2)
 - GP2 = step1 (pin 4)
 - GP3 = dir1 (pin 5)
 - GP4 = step2 (pin 6)
 - GP5 = dir2 (pin 7)
 - GP6 = step3 (pin 9)
 - GP7 = dir3 (pin 10)
 - GP8 = encoder 0 A (pin 11)
 - GP9 = encoder 0 B (pin 12)
 - GP10 = encoder 1 A (pin 14)
 - GP11 = encoder 1 B (pin 15)
 - GP12 = encoder 2 A (pin 16)
 - GP13 = encoder 2 B (pin 17)
 - GP14 = encoder 3 A (pin 19)
 - GP15 = encoder 3 B (pin 20)
